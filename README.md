# devroast

> paste your code. get roasted.

**devroast** é uma aplicação web divertida que analiza seu código e dá feedback — seja honesto ou no modo "roast" (sarcasmo total). Feito durante o **NLW da Rocketseat**.

## O que faz?

- **Cole seu código** — basta colar qualquer código na área de input
- **Escolha o modo** — modo honesto ou modo roast (com máximo de sarcasmo)
- **Veja o resultado** — receba uma análise com feedback sobre seu código
- **Leaderboard** — veja os piores códigos do internet, ranqueados por vergonha

## Funcionalidades

- Interface estilo terminal/IDE
- Destaque de sintaxe para múltiplas linguagens
- Sistema de pontuação de vergonha
- Leaderboard com os piores códigos enviados

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Como rodar

```bash
# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
